# Ảnh dùng cho site

## Menu mobile — 10 ảnh nền (mỗi mục một ảnh)

Trên mobile, mỗi mục trong menu 3 gạch có **một ảnh nền riêng**, hiển thị phía sau với **độ trong suốt 20%**.

| File          | Mục menu    |
|---------------|-------------|
| `menu-1.png`  | Phim bộ     |
| `menu-2.png`  | Phim lẻ     |
| `menu-3.png`  | Thể loại    |
| `menu-4.png`  | Quốc gia    |
| `menu-5.png`  | Danh sách   |
| `menu-6.png`  | Diễn viên   |
| `menu-7.png`  | Hoạt hình   |
| `menu-8.png`  | TV Shows    |
| `menu-9.png`  | Giới thiệu  |
| `menu-10.png` | Donate      |

Đặt 10 file vào thư mục `public/images/`. Nên dùng ảnh đã xóa nền (PNG trong suốt) hoặc ảnh thumb phim phù hợp từng mục.
